RITA DSL

RITA is a DSL designed to create language patterns.
These are used to find and extract information from text, or in other words - NLP

Demo

This is a short demo to show how these patterns looks and works. For full syntax guide, refer to Syntax Documentation

Open Source

RITA Language is an OpenSource project, under MIT license, you can freely use it in your project for both, personal and commercial needs. View it on Github

Plugins

We provide some tools making working with RITA Language easier and more efficient

RITA Language

This is plugin for JetBrains IDEs (IntelijJ, PyCharm, etc.) which adds code highlights and syntax checks. It is totally free.

JetBrains Marketplace

RITA Studio

This is plugin for JetBrains IDEs (IntelijJ, PyCharm, etc.) which adds ability to have live preview of text matches by rules
